### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/EpicCMoment/Projects_Dashboard.git
    cd Projects_Dashboard
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```
